\paragraph{Summary.} I used the \codeword{lex} software package and \codeword{Python3.7} for this assignment.
%
The \codeword{run.sh} script installs the necessary python packages and runs the program on the four sample tests in the \codeword{./tests} folder.
%
\redc{Please replace }\codeword{$python} \redc{in} \codeword{run.sh} \redc{with your python alias.}
%

\paragraph{Compilation.}
%
Run the following command to install the necessary packages for python3

\codeword{local-python -m pip install argparse PrettyTable ply}

\noindent where please replace \codeword{local-python} by your python alias.


\paragraph{Execution.}
%
%

Run

\codeword{python main.py test-file | sort > output-file}

\noindent to execute the program. Optionally, you can pass the \codeword{--pretty} flag for a more readable output

\codeword{python main.py test-file --pretty > output-file}

\noindent Note, that the later command (with \codeword{--pretty}) is not be ideal for checking the output with a \codeword{diff} or \codeword{vimdiff} commands.
